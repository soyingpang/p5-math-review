# 小五數學溫習器

這是一個可離線使用的香港小學五年級數學溫習網頁 App。直接開啟 `index.html` 就可以使用。

## 介面

- 練習台：選單元、切換專題／綜合／錯題模式、即時批改。
- 單元地圖：瀏覽小五 12 個數學單元和範疇分佈。
- 進度報告：查看整體掌握度、各單元準確率和優先重溫項目。

## 範圍

範圍按教育局《數學教育學習領域課程指引補充文件：小學數學科學習內容》（2017）小五單元整理：
https://www.edb.gov.hk/attachment/tc/curriculum-development/kla/ma/curr/pmc2017_tc.pdf

- 5N1 多位數
- 5N2 分數（三）
- 5N3 分數（四）
- 5N4 小數（三）
- 5N5 分數（五）
- 5A1 代數的初步認識
- 5A2 簡易方程（一）
- 5M1 面積（二）
- 5M2 體積（一）
- 5S1 圓
- 5S2 立體圖形（三）
- 5D1 棒形圖（三）

## 使用

1. 用瀏覽器開啟 `index.html`。
2. 選擇左邊單元，按題目作答。
3. 可切換「專題」、「綜合」或「錯題」模式。

目前單元、模式、進度和錯題會存在同一個瀏覽器的 `localStorage`。
