# 核心科溫習器

香港小一至中六中英數溫習網頁 App。可直接用瀏覽器開啟 `index.html`，亦可透過 GitHub Pages 使用。

## 內容

- 科目：中國語文、English、Mathematics
- 年級：小一至小六、中一至中六
- 模式：專題、綜合、錯題
- 分頁：練習台、單元地圖、進度報告
- 題庫：187 個溫習範疇；中英每個範疇有對應題目池，數學按範疇生成不同題型
- 手機：響應式版面、底部快速切換、PWA manifest、Service Worker 離線快取
- 每日目標：可選 5、10、15、20、30 題，完成度存在同一瀏覽器

## 課程來源

範圍參考教育局中國語文、英國語文及數學教育學習領域課程文件整理。數學小五單元按教育局《數學教育學習領域課程指引補充文件：小學數學科學習內容》（2017）細分：
https://www.edb.gov.hk/attachment/tc/curriculum-development/kla/ma/curr/pmc2017_tc.pdf

## 本機使用

直接開啟 `index.html`，或用任何靜態伺服器開此資料夾。

進度、錯題、目前科目、年級、模式、分頁和每日目標會存在同一個瀏覽器的 `localStorage`。
